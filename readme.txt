== Description ==

This plugin will upgrage woocommerce default dropdown to switches. it will uses default image of the variation. 

Links: [Homepage](http://logicdigger.com) | [Donate](http://paypal.me/logicdigger) | [Contact us](https://logicdigger.com/contacts/) | [Support](https://wordpress.org/support/plugin/variation-image-color-switcher-for-woocommerce/) 

== Installation ==

1. Install via the built-in WordPress plugin installer. Or download and unzip `woocommerce-variation-image-color-switcher.zip` inside the plugins directory for your site (typically `wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' admin menu in WordPress
3. When on the dashboard -> settings -> Logicdigger Image Switch, there are settings for styling.
